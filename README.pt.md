# 👨‍💻 Detecção de Faces

Esta aplicação baseada em Python utiliza **OpenCV** e **CVZone** para detectar rostos humanos em tempo real usando a sua webcam.

![Detecção de Faces](https://github.com/KrishBharadwaj5678/FaceDetection/raw/main/FaceDetectionDemo.png)

## 📦 Funcionalidades

| Funcionalidade                         | Descrição                                               |
| -------------------------------------- | ------------------------------------------------------- |
| 🎥 **Detecção de Faces em Tempo Real** | Detecta rostos instantaneamente utilizando a sua webcam |
| 🟥 **Caixa Delimitadora**              | Desenha um retângulo em torno de cada rosto detectado   |
| ⚡ **Otimização da Taxa de Quadros**    | Garante um desempenho suave com monitorização de FPS    |
| 🪶 **Leve e Responsiva**               | Utiliza poucos recursos para uma resposta rápida        |
| 💻 **Código Amigável para Iniciantes** | Simples, legível e fácil de personalizar                |

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia    | Descrição                                                                 |
| ------------- | ------------------------------------------------------------------------- |
| 🐍 **Pitão** | Linguagem de programação de alto nível utilizada para scripts             |

## 🖥️ Instalação

### 1️⃣ Clonar o Repositório

```bash
git clone https://github.com/KrishBharadwaj5678/FaceDetection.git
```

### 2️⃣ Navegar para a Pasta do Projeto

```bash
cd FaceDetection
```

### 3️⃣ Instalar as Dependências

```bash
pip install -r requirements.txt
```

### 4️⃣ Executar o Script

```bash
python main.py
```
