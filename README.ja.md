# 👨‍💻 顔検出

この Python ベースのアプリケーションは、**OpenCV** と **CVZone** を活用して、Webカメラを使用したリアルタイムの顔検出を実現します。

![顔検出](https://github.com/KrishBharadwaj5678/FaceDetection/raw/main/FaceDetectionDemo.png)

## 📦 機能

| 機能                 | 説明                           |
| ------------------ | ---------------------------- |
| 🎥 **リアルタイム顔検出**   | Webカメラを使用して瞬時に顔を検出します        |
| 🟥 **バウンディングボックス** | 検出された各顔の周囲に矩形を描画します          |
| ⚡ **フレームレート最適化**   | FPS の追跡によりスムーズなパフォーマンスを実現します |
| 🪶 **軽量かつ高応答性**    | 最小限のリソースで高速な応答を提供します         |
| 💻 **初心者向けコードベース** | シンプルで読みやすく、簡単にカスタマイズできます     |

## 🖥️ インストール

### 1️⃣ リポジトリをクローン

```bash
git clone https://github.com/KrishBharadwaj5678/FaceDetection.git
```

### 2️⃣ フォルダへ移動

```bash
cd FaceDetection
```

### 3️⃣ 依存関係をインストール

```bash
pip install -r requirements.txt
```

### 4️⃣ スクリプトを実行

```bash
python main.py
```
