# 👨‍💻 Обнаружение лиц

Это приложение на Python использует **OpenCV** и **CVZone** для обнаружения человеческих лиц в режиме реального времени с помощью вашей веб-камеры.

![Обнаружение лиц](https://github.com/KrishBharadwaj5678/FaceDetection/raw/main/FaceDetectionDemo.png)

## 📦 Возможности

| Возможность                               | Описание                                               |
| ----------------------------------------- | ------------------------------------------------------ |
| 🎥 **Обнаружение лиц в реальном времени** | Мгновенно обнаруживает лица с помощью веб-камеры       |
| 🟥 **Ограничивающая рамка**               | Рисует прямоугольник вокруг каждого обнаруженного лица |
| ⚡ **Оптимизация частоты кадров**          | Обеспечивает плавную работу благодаря отслеживанию FPS |
| 🪶 **Лёгкость и отзывчивость**            | Использует минимум ресурсов для быстрой работы         |
| 💻 **Код, удобный для новичков**          | Простой, понятный и легко настраиваемый код            |

| Технология    | Описание                                                                   |

## 🖥️ Установка

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/FaceDetection.git
```

### 2️⃣ Перейдите в папку проекта

```bash
cd FaceDetection
```

### 3️⃣ Установите зависимости

```bash
pip install -r requirements.txt
```

### 4️⃣ Запустите скрипт

```bash
python main.py
```
